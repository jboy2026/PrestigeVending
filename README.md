# Prestige Vending LLC Website

A clean Vite + React landing page for Prestige Vending LLC.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to a GitHub repo.
2. In Vercel, click **Add New Project**.
3. Import the repo.
4. Keep defaults:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`
5. Deploy.

## Update before launch

Replace these placeholders in `src/App.jsx`:

- Phone number
- Email address
- Any real customer testimonials
- Any real photos or logo files
